Assembly Parser

This project is a parser written with JavaScript that can disassemble files written with a subset of ARMv8 Assembly. The design is highly modular and is easily extensible, allowing lots of future expansion. This project earned a perfect grade for a major class assignment.

Goto Language Parser

This was a lexer, parser, and syntax highlighter built in JavaScript for a custom Assemby-like language. It was designed for First Tech Challenge to create an easier and more intuitive way to control the robot.
